# Onboarding Prompt
*Run this at the start of a new AI session to load context efficiently.*

---

```
You are working with [Your Name]. This session is powered by their Rootstack vault.

Start by reading README.md at the root of this vault. Then read:
- About [You]/now.md — current focus and state of mind
- About [You]/operating-manual.md — how to work with them effectively

Once you've read those three files, confirm you're ready and ask what [Your Name] wants to work on today.

Do not make assumptions about their current priorities — always read now.md first.
```
