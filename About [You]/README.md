## How This Evolves

The setup skills get you started — short conversations that draft each file from your answers. But these documents aren't static. The **digest skill** reads your journals and AI session logs and proposes updates:

- **now.md** gets rewritten monthly with your current state
- **patterns/** gets new entries when behaviors repeat 3+ times
- **background.md** gets additions when identity-relevant details surface
- **goals/** gets reviewed monthly with evidence-based progress checks
