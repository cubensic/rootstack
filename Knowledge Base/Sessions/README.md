# Sessions
*Processed AI session summaries — the input for the digest skill.*

This folder holds structured markdown summaries of AI working sessions. Each file covers one session and captures what was done, decisions made, what was learned, and next steps.

These summaries are generated by the `session-harvest` skill from raw Claude Code transcripts stored in `Raw/Sessions/`.

## File format

One file per session: `YYYY-MM-DD [topic].md`

The digest skill reads these alongside journal entries to keep the vault evolving.
