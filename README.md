### `Journal/`
Daily entries. Personal and work reflection are kept separate.

```
Journal/
  Personal/    ← Personal daily entries (DD-MM-YYYY.md)
  Work/        ← Work daily entries (DD-MM-YYYY.md)
```

Use journal entries when you need to understand current state of mind, recent events, or what was happening on a specific date.

---

### `Knowledge Base/`
AI-maintained knowledge base using a three-layer pattern: raw sources (immutable), wiki pages (AI-maintained), and a schema that ties it together.

```
Knowledge Base/
  index.md       ← Master catalog of all KB content
  log.md         ← Append-only record of operations
  Notes/
    work/        ← Wiki pages from work-related sources
    personal/    ← Wiki pages from personal interest sources
    my-content/  ← Copies of published content
  Raw/
    Sessions/    ← Raw .jsonl session transcripts (from Claude Code)
    Sources/     ← Immutable snapshots of ingested articles, papers, etc.
  Sessions/      ← Processed session summaries (markdown, fed into digest)
```

Say "ingest this" to add a source. The AI snapshots it, generates wiki pages, and cross-references everything automatically.
