# Content Pillars
*Strategic approach per platform. One file per platform you publish on.*

Create a file for each of your platforms using this structure:

---

## Template: Platform Pillar

**Platform name:**
**Goal:** [What this platform does for you — discovery, nurture, revenue, etc.]
**Audience:** [Who you're talking to here]
**Format:** [What content looks like on this platform]
**Posting rhythm:** [How often]
**Tone adaptation:** [How your voice shifts for this platform]
**What works:** [Formats and topics that consistently perform well]
**What doesn't:** [What to avoid]

---

Create files like:
- `LinkedIn.md`
- `Newsletter.md`
- `YouTube.md`
- `X (Twitter).md`
