# Content
*Everything related to creating content.*

## Structure

```
Content/
  Style/
    writing-style.md       ← Your voice, tone, and style rules
  Pillars/                 ← Strategic approach per platform
  Schedule/
    LinkedIn/              ← LinkedIn posts (DD-MM-YYYY.md)
    Newsletter/            ← Newsletter issues
    YouTube/               ← Video scripts and notes
    X (Twitter)/           ← Tweet drafts
```

## How It Works

**Style/** is the foundation. Before writing anything, an AI agent should read `writing-style.md` to understand your voice. This is the single most important file for content quality.

**Pillars/** contains one strategy file per platform — audience, goals, format, tone, posting rhythm. Read the relevant pillar file when planning content for a specific platform.

**Schedule/** is where drafts live. Files are named by date (`DD-MM-YYYY Platform.md`). Recent files sit in the platform root folder; older files can be archived in month subfolders.

## Platform Note
The default platforms in Schedule/ are LinkedIn, Newsletter, YouTube, and X. Add or remove folders to match the platforms you actually use.
